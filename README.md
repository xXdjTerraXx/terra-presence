# Terra Presence 🌿

A lightweight desktop tray app built with Electron.

This app allows me to manually update my current activity (music production, gaming, coding, etc.) and sync it with my personal website.

## Tech Stack

- Electron
- Node.js
- Express (backend integration planned)

## Goals

- System tray integration
- Real-time website updates
- Cozy reactive UI
